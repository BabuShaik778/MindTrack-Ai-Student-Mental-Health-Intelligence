MindTrack AI

AI-Powered Student Intelligence Platform for Predictive Mental Health & Academic Risk Analysis

MindTrack AI transforms behavioral and academic data into actionable insights. Using advanced machine learning models, institutions can proactively identify at-risk students and implement early interventions—shifting from reactive systems to predictive intelligence.

📈 Product Evolution

V1.0 – ML Prototype

Built core prediction models and EDA using Google Colab.

V2.0 – Streamlit MVP

Interactive UI for testing and validation.

V3.0 – Flask Production

Scalable architecture with dashboards and deployment readiness.

🤖 ML Pipeline
  • Random Forest for robust predictions
  • Scikit-learn for ML workflows
  • Pandas & NumPy for data handling
  • Joblib for model deployment
  • Custom feature engineering
📊 Analytics

Built using Matplotlib & Seaborn to track behavioral patterns like stress, sleep, and screen time, enabling data-driven insights.

🛠 System Architecture

Frontend Layer: HTML5, CSS3, JavaScript

Backend Framework: Flask (Python Web Framework)

Templating Engine: Jinja2 (Dynamic Rendering)

Machine Learning: Trained ML Models (Inference Engine)

Dashboard: Interactive Data Visualization

Design System: Responsive & User-Centric UX

🎯 Business Impact

Enables institutions to move from reactive support systems to proactive intervention strategies, improving student outcomes and institutional efficiency through predictive analytics.
